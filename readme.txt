Integrate the signup form you created on bullsender.com with your Wordpress site.


== Description ==

After installation acquire your API key on the bullsender site (settings -> user info), and enter it on your Wordpress site.
Once API key is entered, you can place the Bullsender widget on your defined areas, and select the form you want to appear.

For support contact the Bullsender team on bullsender.com.
